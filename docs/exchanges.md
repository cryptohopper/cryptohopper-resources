## How to Connect an Exchange to Cryptohopper

1. **Create an API key** on your chosen exchange. Navigate to the API management section of your exchange account.
2. **Set permissions** to trade-only. Do not enable withdrawal permissions.
3. **Copy the API key and secret** into Cryptohopper's exchange connection settings.
4. **Select your base currency** and configure your bot's trading parameters.

## Important Notes

- **API key security**: Never share your API keys with anyone. Only enter them into the official Cryptohopper platform.
- **Withdrawal permissions**: Cryptohopper does not require withdrawal permissions. Always leave this disabled for maximum security.
- **IP whitelisting**: Some exchanges support IP whitelisting for API keys. Where available, restrict your API key to Cryptohopper's server IP addresses for added security.
- **Exchange availability**: Exchange availability may vary by region due to local regulations. Check your exchange's terms of service for geographic restrictions.

## Spot vs. Futures Trading

- **Spot trading** involves buying and selling actual cryptocurrency assets. You own the coins you purchase.
- **Futures trading** involves trading contracts that derive their value from the underlying cryptocurrency. Futures allow leveraged positions and the ability to profit from both rising and falling markets, but carry higher risk.

Cryptohopper supports both spot and futures trading on exchanges that offer these markets. Check the table above for exchange-specific availability.
